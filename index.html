Call Graph for Fibonacci Sequence:

The fibonacci sequence is represented by the following function: F(n) = F(n-1) + F(n-2) with F(0) = 0 and F(1) = 1

Call Graph for Pell Sequence:

The pell sequence is represented by the following function: a(0) = 0, a(1) = 1; for n > 1, a(n) = 2*a(n-1) + a(n-2)

For more details about the pell sequence, go here to learn more.

Call Graph for Tribonacci Sequence:

The tribonacci sequence is represented by the following function: a(n) = a(n-1) + a(n-2) + a(n-3) with a(0)=a(1)=0, a(2)=1
